A web application built for **Formula 1 fans** to view race schedules, make predictions, and compete on leaderboards.  

---

## 📋 Features  

- **Home Page** – Entry point of the application.  
- **Race Schedules** – Admin creates and manages races.  
- **Predictions** – Users can submit predictions (when enabled by the admin).  
  - **Prediction History** – Review past predictions and outcomes.  
- **Leaderboard** – Displays rankings of all predictions.  
- **About Page** – Explains what the application is about.  
- **Authentication** – Sign in / Sign out button with Google.  

---

## 🛠️ Tech Stack  

- **Frontend:** React+Vite – Responsive and dynamic UI.  
- **Backend:** Firebase  
  - **Authentication** – Social login with Google or email/password.  
  - **Realtime Database** – Store race schedules, predictions, and user data.  
  - **Realtime Syncing** – Updates race and prediction data in real time.  

---

## 🔑 Key Features  

1. **User Authentication** – Secure login with Google via Firebase Authentication.  
2. **Database Management** – Store and retrieve race schedules, predictions, and user IDs using Firebase Realtime Database.  
3. **Real-Time Updates** – Instant syncing of race schedules and predictions.  
4. **Prediction History** – Users can track past predictions and results.  
5. **Admin Control** – Admin manages races and enables/disables predictions.
